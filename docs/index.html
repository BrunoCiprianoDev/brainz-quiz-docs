Sobre o projeto Brainz Quiz

O Braiz Quiz é um projeto desenvolvido com o objetivo de oferecer uma experiência educacional envolvente e interativa para alunos do ensino médio, com foco no aprendizado de química

Sobre esse documento

Este repositório de documentação abrangerá todos os detalhes do projeto, requisitos, padrões de commit, estrutura de banco de dados e diretrizes de implantação.
