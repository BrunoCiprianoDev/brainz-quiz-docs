Requisitos funcionais do sistema [RFs]

Cadastro de perguntasAtualização: 12/10/2023
RF Descrição
RF01 O sistema deve permitir o cadastro de questões com assunto, nivel de dificuldade, alternativas(respostas possiveis)
O sistema deve permitir dois tipos de cadastro de usuário: Administrador e Jogador.Atualização: 12/10/2023
RF Descrição
RF02 Os usuário deve ser cadastrados com: nome, email, password e regra (Administrador, Jogdor) Administrador: Capaz de adicionar novas perguntas, definir pontuações e cadastrar assuntos. Jogador: Pode responder perguntas e acumular pontuações.
O sistema contabilizar a pontuação dos jogadores com base nas questões que acertarAtualização: 12/10/2023
RF Descrição
RF03 O sistema deve calcular a pontuação das perguntas respondidas corretamente, levando em consideração o nível de dificuldade de cada questão.
O sistema deve fornecer feedback imediato ao usuário após responder a uma questãoAtualização: 12/10/2023
RF Descrição
RF04 O sistema deve fornecer feedback imediato ao usuário após responder a uma questão, indicando se a resposta foi correta ou incorreta.
Detalhes do perfil do usuárioAtualização: 12/10/2023
RF Descrição
RF05 O sistema deve oferecer aos usuários a capacidade de visualizar informações pessoais, como nome, e também sua pontuação acumulada em seus perfis.
Recuperação de senhaAtualização: 12/10/2023
RF Descrição
RF06 O sistema deve permitir que o usuário atualize sua senha.
