Tecnologias

Backend

Application Programming Interface

A construção da API foi feita em Node com Typescript, uma plataforma de tempo de execução JavaScript de alto desempenho e escalonável.

Bibliotecas e dependências
Nome da Biblioteca Descrição
Express Ferramenta para do desenvolvimento serviços de API com Node.js
Prisma Ferramenta para o gerenciamento de ORM
Bcryptjs Ferramenta para criptografia e hash para proteção de rotas
Swagger Ferramenta para documentação de Endpoints do serviço de API
Axios Ferramenta para envio de solicitações HTTP a servidores externos
Jsonwebtoken Ferramenta para criptografica de rotas
Multer Ferramenta o envio de arquivos requisições de rotas
Nodemailer Ferramenta para o gerenciamento de e-mails automáticos
Date-fns Ferramenta para manipulação de datas
Cors Ferramenta de segunraça para o controle de acesso aso endpoints

Frontend

Interfaces de usuários

React-Native para o gerenciamento das interfaces das aplicações mobile.

Bibliotecas e dependências
Nome da Biblioteca Descrição
React-Native-Navigation Biblioteca usada para a navegação entre as views
React-Native-Async-Storage Biblioteca usada para armazenar dados na memória do dispositivo

Gerenciamento de dados

Sistema de gerenciamento de banco de dados

Para o gerenciamento de banco de dados, a ferramenta utilizada foi o Postgres 15

Bibliotecas e dependências
Nome da Biblioteca Descrição
Postgres 15 Sistema de gerenciamento de banco de dados
